Tips

  • The JSON on the left represents a database schema, which you can change depending on your project.
  • A JSON table object has a "name" and "fields" properties, you do not need to specify a primary key as we generate it for you.
  • The allowed types for a column are ShortText, LongText,DateTime, Numeric,SingleSelect, MultiSelect. For SingleSelect and MultiSelect you need to specify the relatedTable property.
  • POST request to Backand REST server (see code below).

Code

//need to set request header with auth token
$http({
  method: 'POST',
  url: https://api.backand.com:8080/1/table/config',
  headers: { AppName: appName },
  data:schema //  JSON.stringify of the table object;
});
