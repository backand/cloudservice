Existing Redux App

Follow these steps to integrate Back& with an existing Redux application

Speed up your Redux development with super simple back-end and a complete and scalable gulpjs-based build system that scaffolds the project for you.


Introduction

Follow these steps to install the Back& SDK for Redux. These are also available on the SDK's GitHub page.

Installation

You can install our SDK using either NPM or Yarn:


    $ npm i -S @backand/redux-sdk
    $ yarn add @backand/redux-sdk

Quick Start

Start by using bkndredux to generate the Types, Actions, and Reducers for your Back& objects:


    $ "./node_modules/.bin/bkndredux" --help
    $ "./node_modules/.bin/bkndredux" user obj1 obj2 obj3... -m (thunk/saga)

Note: user is a unique object. It has different Reducer and Types objects, and it reveals most of the authentication Actions (getUserDetails,signin,signout,etc...)

Next, include the Reducers in combineReducers():


    import { combineReducers } from 'redux'
    import user from './user/userReducer'
    import obj1 from './obj1/obj1Reducer'
    import obj2 from './obj2/obj2Reducer'

    combineReducers({
      user,
      obj1,
      obj2
    })

Middleware

To use Redux with either Thunk or Saga as middleware, follow the instructions in the appropriate seciton below.

redux-thunk

Download redux-thunk and include it in createStore()


    import { createStore, applyMiddleware } from 'redux'
    import thunk from 'redux-thunk'

    createStore(rootReducer, initialState, applyMiddleware(thunk))

redux-saga

Download redux-saga and include it in createStore()


     import { createStore, applyMiddleware } from 'redux'
    import createSagaMiddleware from 'redux-saga'
    import rootSaga from './sagas.js'

    const sagaMiddleware = createSagaMiddleware()
    createStore(rootReducer, initialState, applyMiddleware(sagaMiddleware))
    sagaMiddleware.run(rootSaga)

Importing Actions and Finalizing

All that's left is to import the Actions and begin dispatching! This is done using the following code:


    import { getUserDetails, signin, useAnonymousAuth, signout } from './user/userActions'

    store.dispatch(signin(username, password))
    store.dispatch(getUserDetails())
