Ionic starter for mobile

4 easy steps to start mobile ionic project:

We're creating your database and will notify you when it's ready.
In the meantime, you can review below how to integrate your new app with Angular...




  1. Install Ionic framework

    First, install Node.js. Then, install the latest Cordova and Ionic command-line tools.Follow the Android and iOS platform guides to install required platform dependencies

    
  $ npm install -g cordova ionic

    Check Ionic Getting Started for more information and troubleshooting


  2. Start the Project

    Create an Ionic project with Back& inside, includes all Back& dependencies and generic controller

    
  ionic start {{docs.currentApp.Name || 'myApp'}} https://github.com/backand/backand-ionic-starter
  cd {{docs.currentApp.Name || 'myApp'}}

    myApp should be your Back&'s app name


  3. Configure Back&'s REST API

    Back& requires that you include secure tokens for anonymous and sign-up REST API calls 

    
  //Update Angular config section in /www/js/app.js

  .config(function (BackandProvider, $stateProvider, $urlRouterProvider, $httpProvider) {
      BackandProvider.setAppName('{{vm.currentApp.Name || 'Your-App-Name'}}');
      BackandProvider.setSignUpToken('{{vm.keys.signup || 'Your-SignUp-Token'}}');
      BackandProvider.setAnonymousToken('{{vm.keys.anonymous || 'Your-Anonymous-Token'}}');
  })

  4. CRUD your data

    You can access Back&'s REST API endpoints using $http requests from your Ionic Angular project.
    You can duplicate the generic service 'ItemsModel' to manage any object. 

    
  //Manage items object in /www/js/services.js

  .service('ItemsModel', function ($http, Backand) {
      var vm = this;
      //get the object name and optional parameters
      vm.getList = function(name, sort, filter) {
        var params = {
          pageSize: 20,
          pageNumber: 1,
          filter: filter || '',
          sort: sort || ''
        };
        return Backand.object.getList(name, params);
      }
      ...
  })

    5. Done. It's that simple

    Using the above starter code, you can simply build great mobile app using Ionic and Back&.
    Continue now in configuring your model, setup security and add server side code...

    Have fun!
