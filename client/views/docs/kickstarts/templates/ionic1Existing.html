Ionic Quickstart Guide

4 easy steps to include Backand in your Ionic project:

We're creating your database and will notify you when it's ready.
In the meantime, you can review below how to integrate your new app with Angular...




  1. Add the Backand scripts


    Include the following code in your index.html file. Make it above the project's code

    
  <!-- Backand SDK for Angular -->
  <script src="//cdn.backand.net/vanilla-sdk/1.0.9/backand.js"></script>
  <script src="//cdn.backand.net/angular1-sdk/1.9.5/backand.provider.js"></script>

    You can install the SDK locally using bower: bower install backand-angular1-sdk


  2. Configure secure calls to Back&'s REST API


    While Back& uses OAuth2 authentication, we've masked this within the API. Simply use the code below to configure the SDK to speak with your Back& app, and the SDK handles the rest: 

    
  //Update Angular configuration section
  myApp.config(function (BackandProvider) {
      BackandProvider.setAppName('{{vm.currentApp.Name || 'Your-App-Name'}}');
      BackandProvider.setSignUpToken('{{vm.keys.signup || 'Your-SignUp-Token'}}');
      BackandProvider.setAnonymousToken('{{vm.keys.anonymous || 'Your-Anonymous-Token'}}');
  })


  3. Read data from your database


    Back&'s SDK includes several utility functions that encapsulate the HTTP calls used for your REST API. In the following example, we create a function called getList that uses the DataService API endpoint to fetch data from your database, which is returned in JSON format. 

    
  //service.js
  function dataService($http, Backand) {
    var vm = this;
    //get the object name and optional parameters
    vm.getList = function(name, sort, filter, pageSize, pageNumber) {
      var params = {
        pageSize: pageSize || 20,
        pageNumber: pageNumber|| 1,
        filter: filter || '',
        sort: sort || ''
      };
      return Backand.object.getList(name, params);
    }
  }

    We also offer getOne, create, update, and delete functions, giving you the full suite of CRUD functionality in the SDK. On top of these functions, you can also use the SDK to trigger server side actions and manage users. All API calls and expected parameters can be reviewed and tested in Back&'s API Playground.

    4. Done. It's that simple


    Using the four steps above, you can easily integrate Backand with your existing Ionic application, allowing you to build great mobile apps with a full-featured back end. From here, you can configure your data model, user security policies, and more from the Backand App Dashboard.

    Have fun!
