Speed up your AngularJS development with super simple back-end and a complete and scalable gulpjs-based build system that scaffolds the project for you.


Installation

Install node.js. Then sass, gulp and bower, if you haven't yet.
Note: on Windows computers, installing sass requires installing Ruby, as sass is implemented as a ruby gem. Simply visit ruby-lang.org for more info.

    $ gem install sass
    $ npm install -g gulp bower

Clone the backand-angular-kickstart project from github, or download the master release.

    $ git clone https://github.com/backand/angular-kickstart.git
    $ cd angular-kickstart

Install the bower and npm dependencies, then use gulp to run the application in development mode.

    $ npm install
    $ bower install
    $ gulp serve

Run & Deploy

Your application is now ready for development! It is available at http://127.0.0.1:3000.

Every file you add, edit or delete in the /client folder will be detected by gulp, and your application will automatically be reloaded.
When you are ready to build a production release, you can use our pre-built Gulp task to generate a distribution:

    $ gulp serve:dist

This task will lint your code, optimize your assets (css, js, and image files), and run unit tests. Once the task has finished successfully, you can find an optimized, ready-to-deploy version of your project in the /build/dist folder.

To read more about the kickstart app's features, the directory structure, the build system, and more, visit the project's github page.
