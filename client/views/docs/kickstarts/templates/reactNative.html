ReactNative Example

We offer a simple example appliction that demonstrates how to work with Back& in a ReactNative application. Follow the instructions below, or head over to the project on github to get started.

Grab the project

Start by cloning the backand-react-native-example project from GitHub:


    $ git clone https://github.com/backand/react-native-example.git
    $ cd react-native-example

Install dependencies

Next, install the project dependencies using NPM:


    $ npm install

Set up your emulator

Use the instructions on Facebook's Github for the platform you will be working with:
  • iOS - XCode must be installed first
  • Android - you will need to install AndroidStudio and run avd first

Run the app

At this point, all that is left is to start the app for the platform you want:


    $ react-native run-ios
    $ react-native run-android

With that, you're all set!
