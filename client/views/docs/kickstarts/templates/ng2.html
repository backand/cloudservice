Installation

Install Node and NPM, if you have not already done so. To install the backand SDK, use the following npm command:

    $ npm install angular2bknd-sdk --save
Once this is done, you'll also need to install the SDK's dependencies. Do so with the following commands:

    $ npm install @types/node --save-dev
    $ npm install @types/socket.io-client --save-dev

Configuring your Angular 2 app for Backand

Once you've installed the package, you'll need to include it in your application. To do so, start by including Backand in each component in which it will be used:

    import { BackandService } from 'angular2bknd-sdk';

Once you've included the library, you just need to add the service to your object's constructor:

    constructor(private backandService:BackandService){}

Configuring Back&

Back& uses OAuth authenticaation, which requires you to include the authentication token with every HTTP call. Luckily, the Back& SDK takes care of this for you. To connect your app to Back&, use the following code:

    this.backandService.setAppName('your app name');
    this.backandService.setSignUpToken('your backand signup token');
    this.backandService.setAnonymousToken('your backand anonymous token');

If you are developing a mobile app, you'll also want to set the isMobile flag in the SDK:

    this.backandService.setIsMobile(true);


At this point, you're ready to work with Back&!
