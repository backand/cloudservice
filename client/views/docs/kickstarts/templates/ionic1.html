Installation

Note: You can also find this kickstart guide on the Ionic marketplace.
First, install the Ionic framework. Once the Ionic framework is installed you can get the starter app using the following command:

    $ ionic start myApp https://github.com/backand/backand-ionic-starter
    $ cd myApp

Next, run the app using Ionic serve.

    $ ionic serve

This brings up the starter app in a browser window. Log in to the starter app using the default login information:
  • Username: ionic@backand.com
  • Password: backand
At this point, you're done! Enjoy exploring the capabilities offered by our Ionic kickstart app. However, if you are interested in developing something more complex, read on for a quick overview of connecting this starter to a custom Back& application.

Changing the app's back end

Once you've finished exploring the Starter app, you'll want to connect it to a custom application that you control. Below, we'll walk through this process. First, start by creating a new app in the Back& Dashboard.

Once your new app is up, you may want to rebuild the data model. The starter application works well with the default model, so you can skip this step if you choose. Copy and paste the following JSON into your app's data model pane:

    [
      {
        "name": "items",
        "fields": {
          "name": {
            "type": "string"
          },
          "description": {
            "type": "text"
          },
          "user": {
            "object": "users"
          }
        }
      },
      {
        "name": "users",
        "fields": {
          "email": {
            "type": "string"
          },
          "firstName": {
            "type": "string"
          },
          "lastName": {
            "type": "string"
          },
          "items": {
            "collection": "items",
            "via": "user"
          }
        }
      }
    ]

After you've updated your data model, update app.js in the starter application to point at your new Back& app. Change lines 25-27 of app.js to the following:

    BackandProvider.setAppName('<Your Back& App name>');
    BackandProvider.setSignUpToken('<Your app's signup token>');
    BackandProvider.setAnonymousToken('<Your app's anonymous token>');

With that, your application is ready to run against your custom app instance! Review our documentation for guidance on adding new features and capabilities.
