Existing Angular 2 App

Follow these steps to integrate Back& with an existing Angular 2 application

Speed up your AngularJS development with super simple back-end and a complete and scalable gulpjs-based build system that scaffolds the project for you.


Overview

Connecting your application with Back& is as simple as installing our SDK, including it in your project, then configuring the app. Follow the steps below to get your Angular 2 application talking with your back end!

Installation

Our SDK is available on both Yarn and NPM, under the package name @backand/angular2-sdk. To install using NPM, use the following command:


    $ npm i -S @backand/angular2-sdk

To install using Yarn, use the following command:


    $ yarn add @backand/angular2-sdk

Including Back& in your application

To include Back& in your application, add the following import statement:


    import { BackandService } from '@backand/angular2-sdk'

Configuring your application to use Back&

First, register BackandService with NgModule. In app.module.ts (or in the correct location for your app), include the following code:


    @NgModule({
      imports: [ ... ],
      declarations: [ ... ],
      providers: [ BackandService ],
      bootstrap: [ AppComponent ]
    })
    export class AppModule { }

Next, configure your app's connection to Back&. In app.component.ts, or at the appropriate location within your app, add the following code:


    @Component({
      selector: 'my-app',
      template: `<h1>Hello angular2-sdk</h1>
        <h3>{{res}}</h3>`
    })
    export class AppComponent implements OnInit {
      res: string;
      constructor(private backand: BackandService) { }
      getList(): void {
        this.res = 'fetching objects...';
        this.backand.object.getList('users').then((res: any) => {
          this.res = `${res.data.length} objects fetched`;
          console.log(res);
        })
      }
      ngOnInit(): void {
        this.backand.init({
          appName: '{{vm.currentApp.Name || 'Your-App-Name'}}',
          anonymousToken: '{{vm.keys.anonymous || 'Your-Anonymous-Token'}}'
        });
        this.getList();
     }
    }

All set!

At this point, your application is all set to work with Back&! Take a look at our documentation for details on the functions offered by the SDK.
