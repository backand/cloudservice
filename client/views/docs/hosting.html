First Time Installation

The Backand CLI (command-line interface) require Node.js and npm, which can both be installed by following the instructions on https://nodejs.org/. Installing Node.js also installs npm. 


  $ npm install -g backand

  # or use sudo (with caution) if required by your system permissions
  # sudo npm install -g backand

To update previous Backand CLI version run this: 


  $ npm update -g backand

  # or use sudo (with caution) if required by your system permissions
  # sudo npm update -g backand

Deploy your Angular Project

To deploy your local Angular project run this command:


  backand sync --master <masterapptoken> --user <userapptoken> --folder /path/to/project/folder
