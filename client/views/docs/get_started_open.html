Quickstart Guide

4 easy steps to include Backand in your Angular or ionic project:

We're creating your database and will notify you when it's ready.
In the meantime, you can review below how to integrate your new app with Angular...




  1. Add the Backand scripts

    Include the above code in your index.html file. Make it above the project's code

    
  <!-- Backand SDK for Angular -->
  <script src="//cdn.backand.net/backand/dist/1.8.11/backand.min.js"></script>

    You can install the SDK locally using bower: bower install angularbknd-sdk#1.8.10


  2. Script Dependencies

    Add 'backand' to the module dependencies of your angular app

    
  //app.js
  angular.module('YOUR-APP-NAME', ['backand'])

  3. Configure secure calls to Backand's REST API

    Backand uses OAuth2 authentication, which requires that you include the authentication token in every HTTP call 

    
  //Update Angular configuration section
  myApp.config(function (BackandProvider) {
      BackandProvider.setAppName('{{docs.currentApp.Name || 'Your-App-Name'}}');
      BackandProvider.setSignUpToken('{{docs.keys.signup || 'Your-SignUp-Token'}}');
      BackandProvider.setAnonymousToken('{{docs.keys.anonymous || 'Your-Anonymous-Token'}}');
  })


  4. Read data from your database

    You can access Backand's API endpoints using $http requests from your Angular project.
    In the following example, we create a function called getList that uses the DataService API endpoint to fetch data from your database, which is returned in JSON format. 

    
  //dataService.js
  function dataService($http, Backand) {
    var vm = this;
    //get the object name and optional parameters
    vm.getList = function(name, sort, filter) {
      return $http({
        method: 'GET',
        url: Backand.getApiUrl() + '/1/objects/' + name,
        params: {
          pageSize: 20,
          pageNumber: 1,
          filter: filter || '',
          sort: sort || ''
        }
      });
    }
  }

    Using $http you can call to any of the CRUD URLs, trigger server side actions, and manage users. All API calls and expected parameters can be reviewed and tested in Backand's API Playground.

    5. Done. It's that simple

    In the above example, we easily used Backand service, and wrote functionality that can retrieve data from your database.
    Using the above code, you can simply modify the URL endpoints to hit all of the relevant API functionality available from your application's playground - simply copy the URL, adjust the parameters per step 4, and you are set!

    In your Angular application, you are the rock star - Backand is here to help you rise and shine. Have fun!
