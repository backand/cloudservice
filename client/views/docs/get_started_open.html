Quickstart Guide

4 easy steps to include Backand in your Angular or ionic project:

  1. Add the Backand scripts

    First, include Back& in your app's index.html file. Be sure to include it above all of your app's project code includes:

    
      <!-- Backand SDK for Angular -->
      <script src="//cdn.backand.net/backand/dist/1.8.11/backand.min.js"></script>

    You can also install the SDK locally using bower: bower install angular1-sdk#1.8.11


  2. Script Dependencies

    Next, add 'backand' to the module dependencies of your angular app

    
      //app.js
      angular.module('YOUR-APP-NAME', ['backand'])

  3. Configure secure calls to Backand's REST API

    While Backand uses OAuth2 authentication, we've masked this within the API. Simply use the code below to configure the SDK to speak with your Back& app, and the SDK handles the rest: 

    
      //Update Angular configuration section
      myApp.config(function (BackandProvider) {
      BackandProvider.setAppName('{{vm.currentApp.Name || 'Your-App-Name'}}');
      BackandProvider.setSignUpToken('{{vm.keys.signup || 'Your-SignUp-Token'}}');
      BackandProvider.setAnonymousToken('{{vm.keys.anonymous || 'Your-Anonymous-Token'}}');
      })


  4. Read data from your database

    Back&'s SDK includes several utility functions that encapsulate the HTTP calls used for your REST API.
    In the following example, we create a function called getList that uses the DataService API endpoint to fetch data from your database, which is returned in JSON format. 

    
      //dataService.js
      function dataService($http, Backand) {
      var vm = this;
      //get the object name and optional parameters
      vm.getList = function(name, sort, filter) {
      var params = {
        pageSize: 20,
        pageNumber: 1,
        filter: filter || '',
        sort: sort || ''
      };
      return Backand.object.getList(name, params);
      }
      }

    We also offer getOne, create, update, and delete functions, giving you the full suite of CRUD functionality in the SDK. On top of these functions, you can also use the SDK to trigger server side actions and manage users. All API calls and expected parameters can be reviewed and tested in Backand's API Playground.

    5. Done. It's that simple

    In the above example, we easily used Backand service, and wrote functionality that can retrieve data from your database.
    Using the above code, you can simply modify the SDK calls to exercise all of the relevant API functionality available from your application's playground - simply check our documentation for the function information, adjust the parameters per step 4, and you are set!

    In your Angular application, you are the rock star - Backand is here to help you rise and shine. Have fun!
