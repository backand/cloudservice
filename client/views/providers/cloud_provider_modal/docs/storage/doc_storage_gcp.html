Connect to Google Storage

Finding Your 'Private Key' and 'Client Email':

  1. Go to the Google Cloud API Manager and select "Credentials" on the left.
  2. Click on "Create credentials" and select "Service account key".
  3. Select "New service account" in the "Service account" dropdown.
  4. Enter a name for your "Service account name" (e.g. "backand-storage").
  5. Select "Storage" --> "Storage Object Admin".
  6. The "Key type" should be "JSON".
  7. Click on "Create" to create your private key. This creates a keyfile, which should be downloaded to your machine.
  8. Open the key file and copy the 'Private Key' and 'Client Email' fields. Alternatively, you can upload the JSON using Backand's 'Copy Settings' link
