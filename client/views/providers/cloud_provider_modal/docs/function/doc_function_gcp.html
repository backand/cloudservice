Obtaining Google Credentials

Enable the necessary APIs

To execute your functions on the Backand platform, you'll need to enable the Google Cloud Functions API.
This allows Backand to link with Google Cloud and execute your functions. To get started, go to the API dashboard, select your project, and enable 'Google Cloud Functions'.

To obtain your 'private key' and 'client email' entries, follow these steps

  1. Go to the Google Cloud API Manager and select "Credentials" on the left.
  2. Click on "Create credentials" and select "Service account key".
  3. Select "New service account" in the "Service account" dropdown.
  4. Enter a name for your "Service account name" (e.g. "backand-platform").
  5. Select "Project" --> "Owner" as the "Role".
  6. The "Key type" should be "JSON".
  7. Click on "Create" to create your private key. This provides you with your keyfile, which you should download to your local machine.
  8. Open the JSON file is ready, copy the 'Private Key' and 'Client Email' fields. Alternatively, you can simply upload the JSON using the 'Copy Settings' link.
