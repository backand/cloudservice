Working with Fn Project

The Fn Project is an event-driven, open source, Function-as-a-Service compute platform that can be run anywhere in the public, private, or hybrid clouds. Fn Project provides simple-but-powerful extensibility.

The Fn Project is open-source software, allowing you to either maintain a local installation, or re-use an existing instance. To get started, head to the Fn Project's Github page

To start working with Fn Project, copy the Fn server URL, and ensure the server is publicly accessible. If you are working with a server on localhost, you can use a tool like ngrok to create a secure tunnel.

Back&'s Demo Fn Project Server
