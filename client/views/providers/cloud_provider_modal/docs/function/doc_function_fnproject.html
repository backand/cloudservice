Working with Oracle Fn Project

Fn is an event-driven, open source, functions-as-a-service compute platform that you can run anywhere - Public, private and hybrid cloud. Simple yet powerful extensibility.

To install Fn use the project page

To start working with fn project copy the Fn server URL and make sure it is public. When working with localhost you can use ngrok to create a secure tunnels to it.

Back& demo server

For demo purposes we have installed fn project on a public server to be used by everyone:

http://functions.backand.com:8080
Be aware that all the functions on the demo server are public and can be accessed by anyone.

When installing fn project locally setting up the security policy is under your responsibility.
If you would like further assistance in setting up the security policy, please contact us.
