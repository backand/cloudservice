Working with OpenFaas

The OpenFaas is an event-driven, open source, Function-as-a-Service compute platform that can be run anywhere in the public, private, or hybrid clouds. OpenFaas provides simple-but-powerful extensibility.

The OpenFaas is open-source software, allowing you to either maintain a local installation, or re-use an existing instance. To get started, head to the OpenFaas's Github page

To start working with OpenFaas, copy the server URL, and ensure the server is publicly accessible. If you are working with a server on localhost, you can use a tool like ngrok to create a secure tunnel.

Be aware that all the functions on the server are public and can be accessed by anyone.

When installing OpenFaas locally setting up the security policy is under your responsibility.
If you would like further assistance in setting up the security policy, please contact us.
