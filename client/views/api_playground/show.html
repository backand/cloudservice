REST API playground

With Backand's REST API Playground, you simply encapsulate 'CRUD' functions within objecte that represents your database model.
In essence, the objects you create to represent your data in Angular also manipulate that data on the backend.

Just select any of the CRUD functionality with any of the advanced parameters to get the endpoints URL you need in Angular.
In every section click on 'Try it out!' to see the JSON result and the endpoint URL.
